# Callstack

Callstack 是一支由 React Native 专家、核心贡献者以及 Meta 合作伙伴组成的团队，致力于通过开源帮助开发者构建高性能、可扩展的应用程序。

## 关于本指南

优化 React Native 应用不仅仅依赖直觉——它需要深入理解 JavaScript、原生代码以及平台特定的构建流程如何协同工作。本指南提供了一种系统化的方法来诊断和解决性能瓶颈，帮助你开发出启动迅速且始终对用户输入响应迅速的应用。

你将学会如何衡量、解读和优化 “可交互时间”（TTI）与 “每秒帧数”（FPS）——这两项是衡量应用响应速度和平滑度的关键指标。探索提升启动速度、消除卡顿动画、确保 UI 交互流畅的技巧。通过实践性的分析、平台差异理解与最佳实践实现，本指南将帮助你掌握基于数据驱动的优化能力，确保 React Native 应用始终处于最佳性能状态。
